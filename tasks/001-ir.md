# 任务 001：定义工作流 IR 和数据结构

## 目标
在 `compiler` crate 中定义工作流的中间表示（IR），包括指令枚举、值类型和工作流结构。

## 文件
- `compiler/src/ir.rs`（新建）
- `compiler/src/lib.rs`（添加模块声明）

## 具体要求

### 1. 定义 `Value` 类型
使用 `serde_json::Value` 作为动态值类型（类型别名）。

### 2. 定义 `MathOp` 枚举
包含 `Add`, `Sub`, `Mul`, `Div`。

### 3. 定义 `Instruction` 枚举
至少包含以下变体：
- `HttpGet { url: String, headers: Vec<(String, String)>, output_var: String }`
- `SetVariable { name: String, value: Value }`
- `MathOp { op: MathOp, lhs: String, rhs: String, output: String }`
- `Return { value: String }`

### 4. 定义 `Workflow` 结构
包含 `instructions: Vec<Instruction>`。

## 验收标准
- 代码通过 `cargo check`。
- 无 `unwrap()`，所有错误使用 `thiserror`（可先不实现错误，仅定义结构）。
- 添加必要的 `derive`（Debug, Clone, Serialize, Deserialize）。
