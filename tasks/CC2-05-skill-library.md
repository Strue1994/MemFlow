# 任务 CC2-05：技能库与技能发现

## 目标
建立可复用的"技能"（Skill）库，技能是带参数的工作流模板。用户可通过自然语言发现并调用技能，类似 Claude Code 的 `SkillTool`。

## 文件
- `compiler/src/skill_loader.rs`（新建）
- `executor/src/skill_registry.rs`（技能注册表）
- `agent-service/src/skill_manager.ts`（技能管理器）

## 具体要求

### 1. 技能定义
- 技能 = 工作流定义 + 元数据（名称、描述、输入参数 Schema、输出格式、标签）。
- 存储在 SQLite 表 `skills` 中，可导入/导出。

### 2. 技能发现
- Agent 在解析用户请求时，先检索技能库（使用向量相似度或关键词匹配）。
- 若匹配到高相似度技能（>0.85），直接建议用户使用，并询问参数。
- 若无匹配，则回退到动态创建工作流。

### 3. 技能创建与分享
- 用户可将自己常用的工作流保存为技能。
- 支持导出技能文件（JSON），分享给其他人导入。

### 4. 技能市场（可选）
- 提供官方技能仓库，用户可一键安装。

## 验收标准
- 对于常见任务（如"发送周报邮件"），技能命中率 > 80%。
- 使用技能比动态创建节省 50% 的 token 和时间。
