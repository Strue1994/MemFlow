# 任务 E1：联邦学习引擎

## 目标
实现多集群联邦学习，各 MemFlow 实例共享优化模式（不共享原始数据），提升全局优化效果。

## 文件
- `learning-engine/src/federated.rs`（新建）
- `learning-engine/src/federated_coordinator.rs`（中央协调器）
- `executor/src/global_registry.rs`（全局工作流仓库）

## 具体要求

### 1. 本地模式上传
- 每个集群定期（每天）将优化后的工作流模式（节点序列、参数调整）序列化为差分隐私保护的向量。
- 上传到中央协调器（可通过 HTTP 或消息队列）。

### 2. 中央聚合
- 协调器收集各集群的模式，使用联邦平均（FedAvg）或更安全的加密聚合（如 Secure Aggregation）。
- 生成全局模式库，分发给各集群。

### 3. 全局工作流仓库
- 从中央协调器拉取全局模式，存储到本地 `global_patterns` 表。
- Agent 在模式匹配时可选择使用本地模式或全局模式。

### 4. 隐私保护
- 不上传原始日志或用户数据。
- 使用本地差分隐私（`epsilon=1.0`）添加噪声。

## 验收标准
- 两个独立集群通过联邦学习共享优化后，双方工作流成功率均提升 >5%。
- 无法从上传的模式中反推原始数据。
